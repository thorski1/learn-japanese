# Learn Japanese

An interactive terminal game that teaches Japanese from beginner level for English speakers. Built on [quest-engine](https://github.com/thorski1/quest-engine).

## Skill Packs

| Pack | Description |
|------|-------------|
| **Hiragana** | Master the 46 basic hiragana characters |
| **Katakana** | Learn katakana for loanwords and emphasis |
| **Greetings** | Essential Japanese greetings and self-introduction |
| **Numbers** | Counting, counters, time, and money |
| **Food** | Ordering food, tastes, and dining culture |
| **Daily Life** | Morning routines, hobbies, weather, shopping |
| **Travel** | Transportation, directions, hotels, emergencies |
| **Culture** | Festivals, martial arts, tea ceremony, etiquette |

## Install

```bash
pip install -e .
```

## Play

```bash
# Launch the full hub (all 8 packs)
learn-japanese

# Launch a single pack
hiragana-quest
katakana-quest
greetings-jp-quest
numbers-jp-quest
food-jp-quest
daily-jp-quest
travel-jp-quest
culture-jp-quest

# Web mode
learn-japanese --web
```

## Narrator

Your guide is **先生 (Sensei)** — a wise, patient Japanese teacher who will walk you through every step of your Japanese learning journey.
